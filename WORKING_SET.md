# Working Set

Manual tracking for multi-session investigations. Update this when working on complex bugs or research spanning multiple days.

**Update format:**
```markdown
## [Date]: [Investigation topic]

**Goal:** [What are we trying to figure out?]

**Theories:**
1. [Theory 1] — Status: [testing/confirmed/ruled out]
2. [Theory 2] — Status: [testing/confirmed/ruled out]

**Experiments run:**
- [What was tried and what it showed]

**Findings:**
- [What we know so far]

**Next steps:**
- [What to try next]
```

---

*This file is manually maintained. Update it when context gets complex enough to need cross-session tracking.*
